# Signup Form Project

A responsive dark-themed Signup Form built using **HTML** and **CSS**.
The project follows a structured layout and includes proper form validation and semantic HTML tags.

## Features

* Clean signup form UI
* Semantic structure using `header`, `main`, and `footer`
* Input fields with required validation
* Fields included:

  * Full Name
  * Email
  * Password
  * Confirm Password
  * Date of Birth
  * Role selection (Student / Employee)
  * Terms & Conditions checkbox
* Centered layout using Flexbox
* Minimal dark theme design inspired by Figma layout

## Technologies Used

* HTML5
* CSS3

## Form Validation

All fields use the `required` attribute to prevent empty submission and improve user input accuracy.

## Project Structure

```
index.html
style.css
favicon.ico
README.md
```

## How to Run

1. Clone the repository
2. Open `index.html` in your browser
3. Fill the form and test validation

## Purpose
This project was created to practice HTML form structure, semantic tags, and modern layout styling.
